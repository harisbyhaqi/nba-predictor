How it's built

The data

Two seasons of NBA box scores. Each team's last 5 games are averaged — points, shooting percentages, rebounds, assists, turnovers — along with rest days and home/away.

The model

A random forest trained to predict each team's final score. This season's games are weighted more heavily so the model stays current as rosters and form shift.

The odds

Win probabilities come from running each game 10,000 times with realistic score variance added. The percentages reflect how often each team wins across those simulations.
