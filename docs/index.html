How It Works

01

Data Collection

Historical game data is pulled from the NBA Stats API — two full seasons of team-level box scores.

02

Feature Engineering

Rolling 5-game averages for points, FG%, 3P%, rebounds, assists, turnovers. Plus rest days and home/away flag.

03

Random Forest Model

A 200-tree Random Forest regressor is trained to predict each team's point total given their recent form.

04

Monte Carlo Simulation

10 000 game simulations add realistic score variance to produce calibrated win probabilities.
